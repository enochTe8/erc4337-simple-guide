# ERC-4337 (Account Abstraction) — Simple Guide

A beginner-friendly explanation of Account Abstraction on Ethereum.

---

## 🚀 What is Account Abstraction?

Normally, Ethereum users:
- Need ETH to pay gas
- Must manage private keys

Account Abstraction changes this.

👉 It allows smart wallets to act like normal accounts.

---

## 🧠 Simple Explanation

Think of it like this:

Instead of sending a transaction directly, you send a request (UserOperation).

A service called a Bundler processes it for you.

---

## ⚙️ Key Components

### 1. UserOperation
- A request instead of a transaction

### 2. Bundler
- Collects requests and submits them to blockchain

### 3. Paymaster
- Pays gas fees on behalf of users

---

## 🔄 Before vs After

**Before:**
- Need ETH
- Complex wallets
- Poor UX

**After (ERC-4337):**
- Gas can be sponsored
- Smart wallets
- Better UX

---

## 🌍 Real Use Case

Apps can:
- Pay gas for users
- Allow login with email
- Remove seed phrases
